# Personal Expense Tracker – Power BI

## Project Overview
This project is a Personal Expense Tracker dashboard built using Power BI.  
It helps analyze monthly expenses and category-wise spending using interactive visuals.

## Tools & Technologies
- Power BI Desktop
- Power Query
- REST API / Table-based data source
- Data Visualization

## Features
- KPI card showing total monthly expenses
- Category-wise expense breakdown (Pie Chart)
- Interactive visuals
- Clean and minimal dashboard design
